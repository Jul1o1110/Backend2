# PrysmaPH - Backend del Portafolio

Hola! Este es el backend para mi proyecto de portafolio, PrysmaPH. Lo estoy construyendo con Java y Spring Boot como parte de mi curso de Backend 2.

## Descripción del Proyecto

El objetivo de este proyecto es crear la API REST que manejará toda la lógica y la conexión con la base de datos para mi portafolio. Aquí se gestionarán cosas como mis proyectos, mi información de contacto, etc.

## ¿Cómo ejecutar el proyecto?

Una vez que tengas todo configurado, puedes ejecutar el proyecto de dos maneras:

1.  **Desde tu (VS Code):**
    *   Busca la clase `PrysmaPhApplication.java`.
    *   Haz clic derecho y selecciona "Run" o "Ejecutar".

¡Y listo! Si todo salió bien, deberías ver en la consola que la aplicación de Spring Boot ha iniciado correctamente.
